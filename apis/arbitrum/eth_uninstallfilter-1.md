# eth\_uninstallFilter - Arbitrum

### **Parameters**

1. `QUANTITY` - The filter id.

```javascript
params: [
  "0xfe704947a3cd3ca12541458a4321c869"
]
```

### **Returns**

`Boolean` - `true` if the filter was successfully uninstalled, otherwise `false`.
