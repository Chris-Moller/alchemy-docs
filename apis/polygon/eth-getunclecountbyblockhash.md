# eth\_getUncleCountByBlockHash - Polygon

## Parameters

* `DATA`, 32 Bytes - hash of a block.

```javascript
params: [
 '0x17fd2777f5f4f15912a6ea49bcb213c6991ba7ef57cdd64a572f719aa2a46c23'
]
```

## Returns

`QUANTITY` - integer of the number of uncles in this block.
